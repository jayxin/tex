\subsection{直译}

\begin{frame}{直译}
直译：顾名思义就是照字面意思来译。
\begin{itemize}
\item 简捷易行
\item 保存原文的格调和“洋味”
\item 求得译名与电影内容的统一美
\item 有助于不断改进新鲜生动的词和说法
\end{itemize}

\end{frame}
